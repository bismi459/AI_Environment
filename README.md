 Zecpath AI System

Zecpath AI System is a modular and scalable artificial intelligence platform designed to support the complete hiring lifecycle of the Zecpath recruitment product.  
This repository contains independent AI service modules that can later be deployed as separate microservices.
The system covers:
- Resume parsing and ATS screening
- Candidate pre-screening intelligence
- Interview intelligence
- Final scoring and decision support
   
Project Objective

To design a professional AI development environment and repository structure that supports:
- clean architecture
- independent AI services
- logging and testing
- future API and microservice deployment
